# MENTAL MATH

Web application where users add their favorite books, allowing other users to find books to read, and comment on discussion board.

### Technologies 

- Angular
- Node.js

##### Other resources

- Canva.com for icons.

### Installation

##### Requirements

- node.js

##### Launch

In root of project folder:

```console
npm install
npm start
```

Shut down with `Ctrl-C`
